AuthController

@RestController
@RequestMapping(value = ["/api/v1/auth"])
class AuthController(authService: AuthService)(source)

인증 관련 요청을 처리하는 컨트롤러

주요 기능:

  • 사용자 로그인 및 토큰 발급

  • 토큰 갱신

  • 로그아웃 처리

API 엔드포인트:

  • POST /api/v1/auth/login: 로그인

  • POST /api/v1/auth/refresh: 토큰 갱신

  • POST /api/v1/auth/logout: 로그아웃
