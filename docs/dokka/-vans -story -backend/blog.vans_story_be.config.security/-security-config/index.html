SecurityConfig

@Configuration
@EnableWebSecurity
class SecurityConfig(jwtProvider: JwtProvider, userRepository: UserRepository, corsConfigurationSource: CorsConfigurationSource)(source)

Spring Security의 전반적인 보안 정책을 설정하는 @Configuration 클래스입니다.

주요 기능

  • JWT 기반 인증 및 인가 처리

  • 인증이 필요 없는 엔드포인트(로그인, 회원가입, Swagger 등) 화이트리스트 처리

  • CORS 정책 연동

  • 세션 관리 정책(STATELESS)

  • CSRF 보호 비활성화(REST API 특성상)
